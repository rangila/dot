# Description

Fish shell configuration and plugins.

## Install

First, install the [dependencies](#dependencies).

From repository root (e.g.: `~/.dot`)

```bash
mkdir -p $HOME/.config/nvim && stow --dotfiles dot -t $HOME/.config/nvim -d nvim/
```
