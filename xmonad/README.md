# Description

Basic vim configuration without any plugins for quick and dirty use.

## Install

First, install the [dependencies](#dependencies).

From repository root (e.g.: `~/.dot`)

```bash
stow --dotfiles dot -t $HOME -d xmonad
```

## Dependencies

### Common

- rxvt-unicode
- xmonad
- xmonad-contrib
- xmobar
- trayer
- stow
- fonts-hack
